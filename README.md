# React Hook Form

A mini project built with **React**, **TypeScript**, **Vite**, **React Hook Form**, and **Zod** to learn form management, schema validation, reusable form components, and controlled inputs.

## 🚀 Live Demo

[![Live Demo](https://img.shields.io/badge/Live%20Demo-React%20Hook%20Form-blue?style=for-the-badge)](https://mohamadsd2002.github.io/react-hook-form/)

---

## ✨ Features

- Product creation form
- Form validation with Zod
- React Hook Form integration
- Reusable form components
- Custom Select with Controller
- Input, Textarea, Checkbox, Radio Group
- Real-time field watching
- Default form values
- Form reset after submission
- TypeScript support
- Feature-based folder structure

---

## 🛠 Tech Stack

- React 19

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MohamadSD2002/react-hook-form.git
```

Go to the project:

```bash
cd react-hook-form
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
